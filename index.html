The Shin Package manager

Shin is a package manager written in bash, it is the one that I currently use to manage my software. I created the Shin package manager because I have about 6 or 7 package managers installed on my mac and I'm never quite sure wich one to use. Shin combines all these package managers into one. Meaning you just have to remember one command to install a package and Shin will automatically handle which one to install it with. During development time I learnt a lot more about bash scripting and how to use it. I recently added a multi-threading option to the config file.
